# IoT Management System

**English** | [简体中文](./README.zh-CN.md)

C# WinForms admin client for an IoT platform: devices, sensors, monitoring points, check items, staff, live data, SMS alerts, and alarms.

| Platform | URL |
|----------|-----|
| **GitHub** | https://github.com/xiaoyutoucom/internet_of_things_system |

## Features

- Device / sensor / monitoring-point management
- Staff configuration and data monitoring
- SMS notification and alarm information
- Desktop CS admin client

## Tech stack

- C# · WinForms

## Build & run

Open the Visual Studio solution (`.sln`) under this repository, restore NuGet packages, then build and run the main WinForms project.
