# 物联网后台管理系统

[English](./README.md) | **简体中文**

基于 C# WinForms 的物联网平台 CS 端后台：设备、传感器、监测点、检测项、人员配置、数据监测，并包含短信发送、报警信息与实时相关能力。

| 平台 | 地址 |
|------|------|
| **GitHub** | https://github.com/xiaoyutoucom/internet_of_things_system |

## 功能

- 设备、传感器、监测点管理
- 人员配置与数据监测
- 短信通知与报警信息
- 桌面端 CS 管理客户端

## 技术栈

- C# · WinForms

## 编译运行

用 Visual Studio 打开仓库中的解决方案（`.sln`），还原 NuGet 包后生成并运行主 WinForms 项目。
